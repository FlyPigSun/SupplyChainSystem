# 模块说明索引

本目录存放供应链产品管理系统各功能模块的核心逻辑说明文件。

每个模块文件包含：模块职责、核心功能、涉及文件、核心逻辑、数据流、关键设计决策、最近更新。

> 每次 codeReview 后，根据改动更新对应模块文件的「最近更新」章节。

## 公共工具模块

以下模块不直接对应业务功能，被多个业务模块依赖：

| 模块 | 路径 | 说明 |
|------|------|------|
| 价格匹配引擎 | `backend/src/utils/priceMatcher.js` | 4 级优先级匹配 + 口味差异检测 |
| 单位换算 | `backend/src/utils/unitConversion.js` | 重量/容积单位标准化换算 |
| 异步路由包装 | `backend/src/utils/asyncHandler.js` | Express 异步异常捕获 |
| 操作日志工具 | `backend/src/utils/logOperation.js` | 统一日志 INSERT，错误静默处理 |
